**FastSurferRunOnC#**

Короткая C# программа для запуска FastSurfer из Docker через PowerShell.
Часть более общего проекта по внедрению FastSurfer в медтехнику AR-технологией на Unity.

Код проверен, функционирует, как задумано.
Для корректной работы нужен открытый Докер с загруженным контейнером FastSurfer.

Проект выполнен в рамках тестового задания.
