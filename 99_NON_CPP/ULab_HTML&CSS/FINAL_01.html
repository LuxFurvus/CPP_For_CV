IF-ELSE Statements in C++

The if-else statement is a fundamental control structure in C++ programming that allows the execution of different blocks of code based on the evaluation of a condition. It provides a way to control the flow of a program and make decisions based on certain conditions.

The if-else statement is essential for implementing branching logic and creating more complex programs with multiple possible paths of execution.

Structure:

The general syntax of the if-else statement in C++ is as follows:

if (condition1) {
    // code to be executed if condition1 is true
}
else if (condition2) {
    // code to be executed if condition1 is false and condition2 is true
}
else if (condition3) {
    // code to be executed if condition1 and condition2 are false, and condition3 is true
}
else {
    // code to be executed if all conditions are false
}

The condition is enclosed within parentheses and can be any expression that returns a boolean value. The block of code following the if statement is executed if the condition is true. If the condition is false, the block of code following the else statement is executed.

It's important to note that the conditions are evaluated in order, and the first condition that evaluates to true triggers the execution of its corresponding block of code. Subsequent conditions are not evaluated once a true condition is found.

Applications:

The if-else statement is widely used in various programming scenarios, such as:

  • Conditional execution: It allows different blocks of code to be executed based on specific conditions, enabling programs to respond differently to different situations.
  • Input validation: By checking user input against certain conditions, the if-else statement helps ensure that the input meets the required criteria before proceeding with further operations.
  • Decision-making: Using if-else statements, programs can make decisions and take different actions based on the current state of variables or user input.
  • Error handling: If-else statements can be used to handle exceptional situations or error conditions by executing specific error-handling code when necessary.

Code Example:

// Include the standard input/output library for input and output operations
#include <stdio.h>

int main() {
    int age; // Declare an integer variable 'age' to store the user's age

    printf("Enter your age: "); // Display a message to prompt the user for input
    scanf_s("%d", &age); // Read the user's input and store it in the 'age' variable

    if (age < 18) { // Check if the age is less than 18
        printf("You are a minor.\n"); // Display a message for minors
    }
    else if (age >= 18 && age < 65) { // Check if the age is between 18 and 64 (inclusive)
        printf("You are an adult.\n"); // Display a message for adults
    }
    else { // If the age is not less than 18 and not within the adult range
        printf("You are a senior citizen.\n"); // Display a message for senior citizens
    }

    return 0; // Return 0 to indicate successful program execution
}
