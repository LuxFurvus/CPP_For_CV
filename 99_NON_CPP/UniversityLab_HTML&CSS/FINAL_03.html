While & Do-While Loops in C++

Loops in programming languages are control structures that allow the execution of a block of code repeatedly as long as a specified condition is true. In C++, two commonly used loops are the while loop and the do-while loop. They both serve the purpose of repetitive execution but differ in their behavior, especially regarding when the condition is evaluated.

While Loop:

The while loop in C++ continuously executes a block of code as long as the specified condition remains true. The condition is evaluated before each iteration.

Explanation:

  • The while loop syntax begins with the keyword while followed by a condition in parentheses.
  • The loop's body is enclosed in curly braces.
  • Inside the loop, the condition is evaluated. If it's true, the code block executes. After each execution, the condition is checked again. If the condition is false, the loop terminates.

Do-While Loop:

The do-while loop is similar to the while loop, but it evaluates the condition after the execution of the loop's body. This ensures that the loop body executes at least once, regardless of the condition's initial value.

Explanation:

  • The do-while loop syntax starts with the do keyword, followed by the loop's body enclosed in braces.
  • After the body, the while keyword and the condition follow the closing brace. This is the key differenceâ€”the condition comes after the loop body.
  • The loop continues as long as the condition remains true. If the condition is false, the loop terminates.

Key Differences:

  1. Condition Evaluation:
    • While Loop: The condition is evaluated before the first iteration, and if it's false initially, the loop might not execute at all.
    • Do-While Loop: The condition is evaluated after the first iteration, ensuring that the loop body executes at least once.
  2. Usages:
    • While Loop: Used when the number of iterations is unknown or when the loop may not need to execute at all based on the initial condition.
    • Do-While Loop: Suitable when the loop body must execute at least once, irrespective of the condition.

Code Example:

#include <stdio.h>

int main(void) {
    int i = 0; // Initializing variable 'i' for the while loop
    printf("Using while loop: ");
    while (i < 5) {
        printf("%d ", i); // Output the current value of 'i'
        i++; // Increment 'i'
    }
    printf("\n");

    int j = 0; // Initializing variable 'j' for the do-while loop
    printf("Using do-while loop: ");
    do {
        printf("%d ", j); // Output the current value of 'j'
        j++; // Increment 'j'
    } while (j < 5);

    return 0;
}
