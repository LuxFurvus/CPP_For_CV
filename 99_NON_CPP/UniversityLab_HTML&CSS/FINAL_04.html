For Loop in C++

A for loop is a control flow statement in programming that allows you to execute a block of code repeatedly based on a specific condition. It is especially useful when you know the number of iterations required in advance. In C++, the for loop is one of the three looping constructs available, alongside the while loop and the do-while loop.

Syntax:

for (initialization; condition; iteration) {
    // code to be executed
}

Let's break down the different parts of a for loop:

  1. Initialization: This part is used to initialize the loop control variable(s) and is executed only once before the loop starts. It typically involves declaring and assigning a value to the loop control variable(s). For example, int i = 0 initializes an integer variable i with the value 0.
  2. Condition: The condition is checked before each iteration of the loop. If the condition evaluates to true, the loop continues to execute the code block. If the condition evaluates to false, the loop terminates, and control is transferred to the next statement after the loop. For example, i < 10 checks if i is less than 10.
  3. Iteration: After each iteration of the loop, the iteration part is executed. It typically involves incrementing or decrementing the loop control variable(s) to ensure that the loop eventually terminates. For example, i++ increments the value of i by 1.
  4. Code block: The code block is the set of statements that are executed repeatedly as long as the condition remains true. It can contain any valid C++ code, including other control structures, function calls, and variable declarations.

Code Example:

#include <stdio.h>

int main(void) {
    // Example of using the break statement
    printf("Example of using the break statement:\n");
    for (int i = 1; i <= 10; ++i) {
        // Check if the current value of i is equal to 5
        if (i == 5) {
            // If true, print a message and exit the loop using the break statement
            printf("Break statement encountered. Exiting the loop.\n");
            break;
        }
        // Print the current value of i
        printf("%d ", i);
    }

    // Example of using the continue statement
    printf("\n\nExample of using the continue statement:\n");
    for (int i = 1; i <= 5; ++i) {
        // Check if the current value of i is equal to 3
        if (i == 3) {
            // If true, print a message and skip the rest of the loop using the continue statement
            printf("Continue statement encountered. Skipping iteration %d.\n", i);
            continue;
        }
        // Print the current value of i
        printf("%d ", i);
    }

    return 0;
}
